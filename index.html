WIKISMITH

The Streaming Static Site Generator

Static site authoring that doesn't hurt

Wikismith is an open-source static site generator that's designed for fans of gulp and bower. It lets you use your favorite packaging and build tools and favors code over configuration.

You are curious

Watch

You want to get started

Install the Yo Generator

npm install generator-wikismith

Create your project

mkdir my-awesome-site
cd my-awesome-site
yo wikismith

Start the server

gulp

You're new to NPM, Yo, and Gulp

First welcome to a great tool-chain. These tools are making developers lives more fun and easy.

Please checkout these articles:

  • Introduction to Node and NPM

  • Introduction to Yo, Gulp, and Bower

  • Installing Node and NPM

And then your ready to get started.
