Getting Started

After you install npm, yo, and gulp.

npm install generator-wikismith

yo wikismith

gulp

Plain Text

Wikismith content operates on simple plain text files. These plain text files (called pages) are composed of front-matter meta data and markdown.

Example Page

---
title: Congratulations!
subtitle: You have successfully setup your Wikismith project.
created: 2014/06/29 16:40 CST
author: Jeffrey R. Hicks
theme: bs3
---

Wiki

Wikismith provides wiki functionality through wiki links

Wiki links are markdown fragments surrounded in double brackets. Wikismith uses these to auto-generate new pages and render links.

Examples:

[[Most Recent Entry]]

[[Second Blog Entry]]

[[First Blog Entry]]

Examples Rendered:

<ul>
<li> <a href="/most_recent_post">Most Recent Post</a> </li>
<li> <a href="/most_recent_post">Most Recent Post</a> </li><li>
<li> <a href="/most_recent_post">Most Recent Post</a> </li>
</ul>

Blogging

Wikismith provides blogging functionality through index entries

Index entries are markdown fragments surrounded in triple brackets that provide a radically simplified blogging functionality. Wikismith uses these to auto-generate new posts and render post previews that include titles, meta data, excerpts, and permalinks.

Examples:

[[[Most Recent Post]]]

[[[Second Blog Post]]]

[[[First Blog Post]]]

Examples Rendered:

<div class="blog-post">
    <h3 class="blog-post-title">Most Recent Post</h3>
    <p class="blog-post-meta">2014/08/04 by J. Hicks</p>
    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Mauris vel ante enim ...</p>
    <p><a href="/most_recent_post">Read More</a></p>
</div>

Formatters

Wikismith formats index entries via formatters. Below is the default formatter for index_entries.

formatters/index_entry.html

<div class="blog-post">
    <h3 class="blog-post-title"><%=params.title%></h3>
    <p class="blog-post-meta"><%=params.created%> by <%=params.author%></p>
    <p><%=params.excerpt%></p>
    <p><a href="<%=url%>">Read More</a></p>
</div>
